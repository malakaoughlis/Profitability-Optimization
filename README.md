# Profitability Optimization - Business Intelligence Case Study

## Project Overview

This project analyzes the Global Superstore dataset to understand why sales growth does not always lead to higher profitability.

The goal is to identify the main drivers of profit and provide business recommendations using Python, SQL, Excel, and Power BI.

## Business Question

How can Nova Retail improve profitability without simply increasing sales?

## Source

https://www.kaggle.com/datasets/fatihilhan/global-superstore-dataset?resource=download
